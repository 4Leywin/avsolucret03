# AV SOLUCRET

Web hecha por Klein Code: https://kleincode.studio/

## Cómo ejecutar localmente

Requisitos: Node.js y npm o pnpm instalados.

Pasos rápidos:

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

## Tecnologías

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

Si necesitas desplegar, usa tu proveedor preferido (Vercel, Netlify, etc.).
